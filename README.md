# SEO

## SEO是什么？

seo是一个终端命令行工具，由于在编译文件报错时为你提供解决方案，实现更方便的面向搜索引擎编程 ：)

灵感来自于 [shobrook/rebound](https://github.com/shobrook/rebound)

觉得挺有意思，就自己整了个国内低配版，后续可能会进一步升级(~~看心情~~)

## 功能

- [x] 编译python
- [x] 编译C
- [x] 编译JAVA
- [x] 支持CSDN
- [x] 支持segmentfual
- [ ] 支持知乎
- [ ] 搭建终端用户界面
